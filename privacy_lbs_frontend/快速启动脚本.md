# 快速启动脚本

## Windows PowerShell 快速启动

### 启动后端（新终端窗口）

```powershell
# 终端1：启动Django后端
cd D:\Search\privacy_lbs_backend
venv\Scripts\activate
python manage.py runserver

# 终端2：启动Celery Worker（如需要）
cd D:\Search\privacy_lbs_backend
venv\Scripts\activate
python -m celery -A privacy_lbs_backend worker --loglevel=info --pool=solo
```

### 启动前端（新终端窗口）

```powershell
# 终端3：启动前端
cd D:\Search\privacy_lbs_frontend
npm run dev
```

## 一键启动脚本（Windows）

创建 `start_all.bat` 文件：

```batch
@echo off
echo 启动隐私保护LBS系统...

echo.
echo [1/3] 启动Django后端...
start "Django Backend" cmd /k "cd /d D:\Search\privacy_lbs_backend && venv\Scripts\activate && python manage.py runserver"

timeout /t 3 /nobreak >nul

echo.
echo [2/3] 启动Celery Worker...
start "Celery Worker" cmd /k "cd /d D:\Search\privacy_lbs_backend && venv\Scripts\activate && python -m celery -A privacy_lbs_backend worker --loglevel=info --pool=solo"

timeout /t 3 /nobreak >nul

echo.
echo [3/3] 启动前端...
start "Frontend" cmd /k "cd /d D:\Search\privacy_lbs_frontend && npm run dev"

echo.
echo 所有服务已启动！
echo 前端地址: http://localhost:5173
echo 后端地址: http://127.0.0.1:8000
echo.
pause
```

## 一键启动脚本（Linux/Mac）

创建 `start_all.sh` 文件：

```bash
#!/bin/bash

echo "启动隐私保护LBS系统..."

# 启动Django后端
echo "[1/3] 启动Django后端..."
cd /path/to/privacy_lbs_backend
source venv/bin/activate
python manage.py runserver &
DJANGO_PID=$!

sleep 3

# 启动Celery Worker
echo "[2/3] 启动Celery Worker..."
cd /path/to/privacy_lbs_backend
source venv/bin/activate
celery -A privacy_lbs_backend worker --loglevel=info &
CELERY_PID=$!

sleep 3

# 启动前端
echo "[3/3] 启动前端..."
cd /path/to/privacy_lbs_frontend
npm run dev &
FRONTEND_PID=$!

echo ""
echo "所有服务已启动！"
echo "前端地址: http://localhost:5173"
echo "后端地址: http://127.0.0.1:8000"
echo ""
echo "按 Ctrl+C 停止所有服务"

# 等待中断信号
trap "kill $DJANGO_PID $CELERY_PID $FRONTEND_PID; exit" INT TERM
wait
```

使用前需要添加执行权限：
```bash
chmod +x start_all.sh
```

## 验证服务运行

### 检查后端
```bash
curl http://127.0.0.1:8000/api/
```

### 检查前端
打开浏览器访问：`http://localhost:5173`

## 停止服务

### Windows
- 关闭各个终端窗口
- 或使用 `Ctrl+C` 停止

### Linux/Mac
```bash
# 查找进程
ps aux | grep "python manage.py runserver"
ps aux | grep "celery"
ps aux | grep "vite"

# 停止进程
kill <PID>
```
