# 启动Vue项目

## 错误原因

错误信息：`Could not read package.json: Error: ENOENT: no such file or directory`

**原因：** 在错误的目录下运行了 `npm run dev`。

- ❌ 错误：在 `D:\Search` 目录下运行
- ✅ 正确：在 `D:\Search\privacy_lbs_frontend` 目录下运行

## 正确的启动方式

### 方法1：切换到项目目录后启动

```powershell
# 1. 切换到项目目录
cd D:\Search\privacy_lbs_frontend

# 2. 启动开发服务器
npm run dev
```

### 方法2：一行命令

```powershell
cd D:\Search\privacy_lbs_frontend; npm run dev
```

### 方法3：创建启动脚本

创建 `start_dev.bat` 文件：
```batch
@echo off
cd /d %~dp0
npm run dev
pause
```

然后双击运行即可。

## 验证项目目录

确保当前目录包含以下文件：
- ✅ `package.json`
- ✅ `vite.config.ts`
- ✅ `src/` 目录
- ✅ `node_modules/` 目录

如果缺少 `node_modules`，需要先安装依赖：
```powershell
cd D:\Search\privacy_lbs_frontend
npm install
```

## 启动成功标志

启动成功后，应该看到类似输出：
```
  VITE v7.x.x  ready in xxx ms

  ➜  Local:   http://localhost:5173/
  ➜  Network: use --host to expose
```

然后在浏览器中访问 `http://localhost:5173/`

## 常见问题

### Q: 仍然报错 "Could not read package.json"
**A:** 检查：
1. 是否在 `privacy_lbs_frontend` 目录下
2. `package.json` 文件是否存在
3. 使用 `pwd` (PowerShell: `Get-Location`) 查看当前目录

### Q: 报错 "command not found: npm"
**A:** Node.js 未安装或未添加到PATH，需要：
1. 安装 Node.js: https://nodejs.org/
2. 重启终端

### Q: 端口被占用
**A:** 修改端口：
```powershell
npm run dev -- --port 3000
```
