# 前端项目结构说明

本文档说明已实现的前端项目结构，按照设计图完成所有模块。

## 📁 项目结构

```
privacy_lbs_frontend/src/
├── types/                    # TypeScript类型定义
│   └── index.ts             # 所有接口和类型定义
├── utils/                    # 工具函数
│   ├── crypto.util.ts       # 加密工具（前端非敏感操作）
│   └── coordinate.util.ts   # 坐标转换工具
├── services/                 # API服务层
│   ├── api.client.ts        # Axios客户端配置
│   ├── query.service.ts     # 查询相关API
│   └── data.service.ts      # 数据上传相关API
├── stores/                   # Pinia状态管理
│   ├── map.store.ts         # 地图状态管理
│   ├── query.store.ts       # 查询状态管理
│   └── result.store.ts      # 结果状态管理
├── components/               # 组件
│   ├── map/                 # 地图相关组件
│   │   ├── SecureMap.vue           # 核心地图容器
│   │   ├── QueryPointLayer.vue    # 查询点图层
│   │   ├── ResultMarkerLayer.vue  # 结果标记图层
│   │   ├── TrajectoryLayer.vue    # 轨迹图层
│   │   └── SafeZoneLayer.vue      # 安全区域图层
│   ├── query/               # 查询相关组件
│   │   ├── QueryForm.vue          # 查询表单
│   │   ├── WeightSlider.vue       # 权重滑块
│   │   └── RealTimeUpdate.vue     # 实时更新组件
│   └── results/             # 结果相关组件
│       ├── ResultList.vue          # 结果列表
│       ├── ResultItem.vue         # 结果项
│       └── RouteDetails.vue       # 路线详情抽屉
├── views/                    # 页面视图
│   ├── HomeView.vue         # 主界面（地图+侧边栏）
│   └── DashboardView.vue    # 数据仪表板（可选）
├── router/                   # 路由配置
│   └── index.ts
├── config/                   # 配置文件
│   └── env.ts               # 环境变量配置
├── App.vue                   # 根组件
└── main.ts                   # 入口文件
```

## 🎯 核心功能特性

1. **连续查询请求** - 支持多次查询和实时更新
2. **实时位置更新** - 自动获取位置并更新查询
3. **文本-距离权重调整** - 动态调整查询权重
4. **结果解密与展示** - 显示查询结果和评分
5. **Top-K推荐结果** - 显示Top-K个最佳结果
6. **路线规划可视化** - 路线详情和地图显示
7. **地图交互** - 完整的地图功能（标记、图层、轨迹等）
8. **结果排序和筛选** - 多维度排序和关键词筛选

## 🔧 技术栈

- **Vue 3** - 前端框架
- **TypeScript** - 类型安全
- **Pinia** - 状态管理
- **Vue Router** - 路由管理
- **Element Plus** - UI组件库
- **Mapbox GL JS** - 地图库
- **Axios** - HTTP客户端
- **Day.js** - 日期处理

## 📝 注意事项

1. **环境变量配置**：需要在 `.env.development` 中配置：
   - `VITE_MAPBOX_ACCESS_TOKEN` - Mapbox访问令牌
   - `VITE_API_BASE_URL` - 后端API地址
   - `VITE_WS_URL` - WebSocket地址

2. **后端API接口**：确保后端实现了以下接口：
   - `/api/secure_query/submit/` - 提交查询
   - `/api/secure_query/{id}/update/` - 更新查询
   - `/api/secure_query/{id}/status/` - 查询状态
   - `/api/secure_query/{id}/cancel/` - 取消查询
   - `/api/secure_query/history/` - 查询历史
   - `/api/secure_query/route/` - 路线规划
   - `/api/data_owner/upload/` - 数据上传
   - `/api/data_owner/upload/{id}/status/` - 上传状态
   - `/api/data_owner/statistics/` - 数据统计

3. **待完善功能**：
   - DashboardView中的图表实现（ECharts集成）
   - 路线在地图上的折线绘制
   - WebSocket实时通信集成
   - 数据上传进度可视化

## 🚀 启动项目

```bash
cd privacy_lbs_frontend
npm install
npm run dev
```

项目将在 `http://localhost:5173` 启动。
