# 清除 Vite 缓存并重启

## 问题原因

Vite 缓存了错误的 axios 模块导入。需要清除缓存并重启。

## 解决步骤

### 方法1：清除 Vite 缓存（推荐）

```powershell
# 停止开发服务器 (Ctrl+C)

# 删除 Vite 缓存目录
Remove-Item -Recurse -Force node_modules\.vite

# 重新启动
npm run dev
```

### 方法2：完全清除并重新安装

如果方法1不行，尝试：

```powershell
# 停止开发服务器 (Ctrl+C)

# 删除所有缓存和依赖
Remove-Item -Recurse -Force node_modules
Remove-Item -Recurse -Force node_modules\.vite

# 重新安装依赖
npm install

# 重新启动
npm run dev
```

## 验证修复

重启后，控制台应该：
- ✅ 不再有 axios 导入错误
- ✅ 能看到 "App 根组件已挂载" 等日志
- ✅ 页面正常显示
